## 环境需求

最低支持 iOS 8.0，推荐 Swift 语言。

## 安装集成

可以通过 [CocoaPods](http://cocoapods.org) 快速集成到项目中，也可以直接下载 [XZKit.framework](./Products) 导入到项目中。

```ruby
pod "XZKit"
```
